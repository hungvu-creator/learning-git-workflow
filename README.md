# learning-git-workflow

6-day project with GPT for understanding SDLC from a developer lens.

## Milestone Checklist

- [ ] Day 1: Install tools and clone repo
- [ ] Day 2: Create branch and make commits
- [ ] Day 3: PR review workflow + Jira integration
  - [ ] Open a Pull Request for a feature branch
  - [ ] Simulate a code review (add comments, request changes, approve)
  - [ ] Connect VS Code / workflow to Jira
  - [ ] Create a Jira item for Day 3 with the following acceptance criteria:
    - [ ] A feature branch has been created and pushed
    - [ ] A Pull Request is opened and linked to the Jira item
    - [ ] At least one review comment has been added and resolved
    - [ ] PR is approved and merged into the main branch
    - [ ] Jira item status is updated to Done
- [ ] Day 4: Practice GitHub Copilot usage
- [ ] Day 5: Build or try a simple AI agent
- [ ] Day 6: Connect VS Code to GitLab and review a larger codebase plus merge requests
